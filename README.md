\# WarehouseIQ — Warehouse Inventory Management System



A full-stack mobile application for managing warehouse inventory, built with React Native (Expo), Node.js, Express.js, and MongoDB.



\## Tech Stack



\- \*\*Frontend:\*\* React Native (Expo)

\- \*\*Backend:\*\* Node.js + Express.js

\- \*\*Database:\*\* MongoDB (Atlas)

\- \*\*File Storage:\*\* Cloudinary

\- \*\*Hosting:\*\* Render

\- \*\*Testing:\*\* Jest + Supertest

\- \*\*Containerization:\*\* Docker



\## Features



\- User authentication (JWT) with role-based access (Admin/Staff)

\- 6 core modules: Products, Suppliers, Warehouses, Stock Entries, Sales Records, Customers

\- Automatic inventory tracking (stock entries increase quantity, sales decrease it)

\- File/image uploads via Cloudinary

\- Search, filter, sort, and pagination across all modules

\- Dashboard with summary statistics and low-stock alerts

\- Clean, warehouse-themed UI
